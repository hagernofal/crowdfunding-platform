\# Crowdfunding Platform – JavaScript Project

\# Project Overview

A full-stack crowdfunding platform built using:



\- HTML5  

\- CSS3  

\- JavaScript (ES6+)  

\- JSON Server  

\- JSON Server Auth  



The platform simulates a real crowdfunding system where:



\- Users can create and support campaigns  

\- Admins can moderate users and campaigns  

\- Anonymous visitors can browse approved campaigns  



\##User Roles



\#Admin

\- Manage users

\- Approve or reject campaigns

\- Delete violating campaigns



\#Registered User

\- Create campaigns

\- Support campaigns via pledges

\- View pledge history



\#Anonymous User

\- Browse approved campaigns only



---



\##Features



\- Authentication system

\- Campaign management

\- Role-based dashboard logic

\- REST API integration using fetch()
